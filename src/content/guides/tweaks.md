## Options

Enable **Options → Show Advanced Options** first.

### General

| Option | Value | Purpose |
|---|---|---|
| Display → Frame rate | Set to Unlimited | Remove FPS cap |
| Automation → Automatically disassemble scrap | Check | Reduces inventory management overhead (you may need to keep the scrap for selling) |
| Automation → Maximum automove cells/sec and autoattack actions/sec | Set to Unlimited | Faster auto-explore |
| Autoget → Autoget books | Check | Almost always worth collecting for EXP or money (you may need to see the book titles before picking them up) |
| Debug → Don't delete Classic mode saved games on death | Check | Keep failed run saves for learning |
| Debug → Show reputation with a creature's faction when looking at them | Check | You will see an exact reputation when looking at ritualable creatures |

### Display & UI

| Setting | Value | Purpose |
|---|---|---|
| Display → Fullscreen resolution | Set to Screen | Fixes fullscreen scaling issues, especially on high-DPI displays |
| Display → Modern and overlay control scale | auto ×1.25 or desired scale | Adjusts UI elements to readable size |
| Display → Play area scale | Pixel Perfect | Eliminates blurriness and scaling artifacts |
| Display → Pixel perfect tile scale | Based on your display (typically 2 for high-DPI) | Ensures crisp tile rendering |

## Keybinds

### Basic Move

| Action | Key | Purpose |
|---|---|---|
| Move (all directions) | <span class="ent-info">Numpad</span> or <span class="ent-info">Arrow keys</span> (<span class="ent-info">Shift+Arrow</span> for diagonal) | Navigate in eight directions |
| Force attack in a direction | <span class="ent-info">`\`</span> | Forcefully attack an object, friend, wall, or anything else |
| Move up | <span class="ent-info">`Shift+,`</span> | Navigate to higher elevations |
| Move down | <span class="ent-info">`Shift+.`</span> | Descend to lower elevations |

### Menus

| Action | Key | Purpose |
|---|---|---|
| Take all | <span class="ent-info">`Tab`</span> | Take everything from a chest, dropped enemy, table, etc |
| Store items | <span class="ent-info">`F2`</span> | Store items in a chest |

### Adventuring

| Action | Key | Purpose |
|---|---|---|
| Look | <span class="ent-info">`L`</span> | Inspect an enemy or object |
| Get | <span class="ent-info">`G`</span> | Pick up item from ground or chest |
| Move to point of interest | <span class="ent-info">`Backspace`</span> | Auto-navigate to a point of interest |
| Interact nearby | <span class="ent-info">`Ctrl+Space`</span> / <span class="ent-danger">`Shift+E`</span> | Interact with objects, NPCs, or features in your immediate vicinity |
| Fire missile weapon | <span class="ent-info">`F`</span> | Launch a ranged attack |
| Reload missile weapon | <span class="ent-info">`R`</span> | Reload with ammunition |
| Throw | <span class="ent-info">`T`</span> | Throw items or weapons |
| Wait | <span class="ent-info">`Shift+Space`</span> | Pause for a turn |
| Wait a number of turns | <span class="ent-info">none</span> / <span class="ent-danger">`Shift+W`</span> | Wait for a specified number of turns |
| Rest until healed | <span class="ent-info">`` ` ``</span> | Remain stationary until full health |
| Autoexplore | <span class="ent-info">`0`</span> (numpad) / <span class="ent-danger">`.`</span> | Automatically explore unexplored areas |

### Character Sheet

| Action | Key | Purpose |
|---|---|---|
| Open abilities | <span class="ent-info">`A`</span> | Access your character's special abilities menu |
| Open inventory | <span class="ent-info">`I`</span> | View and manage your items and equipment |

### Trade

| Action | Key | Purpose |
|---|---|---|
| Offer | <span class="ent-info">`O`</span> | Initiate a trade by offering items to merchants |
| Ask vendor to identify | <span class="ent-info">`I`</span> | Request a vendor to identify artifacts |

> All menus and interactions support both hotkeys and mouse control. When using hotkeys (e.g. `I` for inventory), menus open with multiple tabs that you can navigate via keyboard or mouse click. Custom keybindings show in <span class="ent-danger">orange</span>, while defaults appear in <span class="ent-info">blue</span>.
